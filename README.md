# Coffee Shop Simulator

## Description
Simple Java simulation of customers and baristas interacting with a coffee shop queue. This project simulates a coffee shop where `Customer` and `Barista` threads interact with a `CoffeeShop` instance. The main entry is `src/main/CoffeeShopSimulator.java`. This project was developed as part of the Concurrent Programming module at the University of Westminster by Tharin Mahale as part of the BEng (Hons) Software Engineering degree program. The learning objective was to understand and implement multithreading concepts in Java, specifically Java Locks.

## Requirements
- Java 11+ (or the Java version configured in your IDE)
- IntelliJ IDEA (recommended for running and debugging)

## Run
- IntelliJ: Open the project, locate `CoffeeShopSimulator.main` and run it.
